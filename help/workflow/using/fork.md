# 포크{#fork}

포크를 사용하면 여러 활동을 동시에 실행할 수 있습니다.

그래픽 개체를 두 번 클릭하여 아웃바운드 전환 수를 정의하고, 새 전환을 만들거나, 선택한 전환의 레이블을 변경합니다.
