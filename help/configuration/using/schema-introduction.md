# 스키마 참조 {#schema-reference}

스키마를 편집할 때 소스 스키마(xtk:srcSchema)를 기반으로 하는 승인 시스템을 사용할 수 있습니다. &quot;데이터베이스 구조 업데이트...&quot;를 사용하여 데이터베이스를 업데이트할 때도 일부 오류가 발견될 수 있습니다. 비서.
