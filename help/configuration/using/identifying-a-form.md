# 양식 식별{#identifying-a-form}

입력 양식을 사용하면 Adobe Campaign 클라이언트 콘솔에서 데이터 스키마에 연결된 인스턴스를 편집할 수 있습니다.

양식은 해당 이름과 네임스페이스로 식별됩니다.

양식의 ID 키는 네임스페이스로 구성된 문자열 및 콜론으로 구분된 이름(&#39;:&#39;)입니다(예:&quot;cus:contact&quot;).
