# 복원{#restoration}

깨끗한 서버에서 복원 절차는 다음과 같습니다.

* 을(를) 클릭합니다.
* 타사 애플리케이션 설치:웹 서버, JDK(필요한 경우),
* 소스 시스템과 동일한 빌드로 Adobe Campaign 바이너리 설치,
* 구성 파일 복사, 추적 로그 및 리디렉션 파일
* 데이터베이스를 만들고 다시 빌드합니다.
* adobe campaign을 시작합니다.

자세한 내용은 **설치 안내서**&#x200B;를 참조하십시오.
