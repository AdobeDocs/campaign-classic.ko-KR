# Adobe에 요청 제출{#submitting-request-to-adobe}

>[!NOTE]
>
>계속 진행하려면 시스템에서 IMS가 활성화되어 있는지 확인하십시오. IMS에 대한 섹션을 [참조하십시오.](../../integrations/using/about-adobe-id.md).

IMS가 활성화되면 Adobe Campaign과 People 코어 서비스/Audience Manager 통합 제공을 요청할 수 있습니다. 이렇게 하려면 Digital-Request@adobe.com으로 이메일을 보내 [다음](mailto:Digital-Request@adobe.com) 정보를 확인하십시오.
